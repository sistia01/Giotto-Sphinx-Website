.. _removeGiottoEnvironment: 

###############################
Remove the Giotto Environment
###############################

.. describe:: removeGiottoEnvironment()

*Removes the Giotto environment*

.. code-block::

	removeGiottoEnvironment(verbose = TRUE)


**********************
Arguments
**********************

.. list-table::
	:widths: 100 100 
	:header-rows: 0 

	* - **verbose**	
	  - be verbose

***************
Details
***************

Removes a previously installed Giotto environment (see `installGiottoEnvironment() <installGiottoEnvironment>`_).
