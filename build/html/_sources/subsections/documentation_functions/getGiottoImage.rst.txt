Get Giotto Image from Giotto Object
#####################################

.. describe:: getGiottoImage()

*Get a giotto image from a giotto object.*

.. code-block::

	getGiottoImage(gobject, image_name)

**********************
Arguments
**********************

.. list-table::
	:widths: 100 100 
	:header-rows: 0 

	* - **gobject**	
	  - giotto object
	* - **image_name**	
	  - name of giotto image `showGiottoImageNames <showGiottoImageNames>`_

******************
Value 
******************
A Giotto image
