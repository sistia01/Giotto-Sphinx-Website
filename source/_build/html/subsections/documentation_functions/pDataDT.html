Show Cell Metadata¶

pDataDT()

Show cell metadata.

pDataDT(gobject)

Arguments¶

gobject

giotto object

Value¶

A data.table with cell metadata

Examples¶

data(mini_giotto_single_cell) # loads existing Giotto object
pDataDT(mini_giotto_single_cell)
