Compute Highly Variable Genes¶

anndataToGiotto()

Compute highly variable genes.

anndataToGiotto(
        anndata_path,
        metadata_cols = c("total_counts", "pct_counts_mt"),
        instructions = NULL,
        ...
)

Arguments¶

anndata_path

path to the .h5ad file

metadata_cols

metadata columns to include

instructions

giotto instructions

…

additional parameters to createGiottoObject()

Value¶

A Giotto object.

Details¶

Converts a .h5ad file into a Giotto object.

Important

Function in beta.
