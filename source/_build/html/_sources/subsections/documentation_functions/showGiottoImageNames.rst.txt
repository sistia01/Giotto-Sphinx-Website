.. describe:: showGiottoImageNames()

*Adds cell metadata to the Giotto Object.*

.. code-block::

	showGiottoImageNames(gobject, verbose = TRUE)

**********************
Arguments
**********************

.. list-table::
	:widths: 100 100 
	:header-rows: 0 

	* - **gobject**	
	  - a giotto object
	* - **verbose**	
	  - verbosity of function

******************
Value 
******************
A vector of Giotto image names attached to the Giotto object
